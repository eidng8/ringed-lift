## 3.1 卷首说明

本卷定位为太空环梯工程的**地面-轨道连接基础**，聚焦于回答以下核心问题：

> **6座太空电梯如何锚固在地球表面？节点与环主体的机械连接如何实现？**

本卷承接第一卷的轨道力学环境（摄动载荷、电梯缆绳独立工况）和第二卷的材料参数（CNT缆绳工程强度、锚碇材料性能），为后续的电梯爬升器与缆绳设计卷（第四卷）和综合物理论证卷（第五卷）提供节点锚碇系统的边界条件。

本卷的任务是：
1. 完成6个地面节点的详细工程地质评估与锚碇方案设计。
2. 输出节点锚碇处位移约束类型与刚度，供第五卷模态分析使用。
3. 给出环段弧距分布、环-节点机械连接接口的完整设计。

本卷不涉及电梯缆绳的张力分布和配重系统（由第四卷承担）、轿厢设计（由第四卷承担）、环主体缆绳的铺设（由第六卷承担）或经济成本核算（由第八卷承担）。凡当前阶段无法确定的参数，均标注“待验证”并给出验证路径。

## 3.2 地面节点选址与工程地质评估

### 3.2.1 选址的工程约束

1. **纬度约束**：锚碇点应位于赤道±1°以内（约±111 km）。
2. **地质约束**：锚碇区须有稳定基岩或可贯入吸力锚的软岩/沉积层。避开活跃火山、强烈地震带（PGA宜<0.3g）和液化风险区。
3. **施工可达性**：须有深水港、国际机场和基本道路/电力基础设施。
4. **长期安全性**：数百年设计寿命内，避开预计会因海平面上升、强烈地质活动而失效的场址。
5. **土地面积约束**：锚碇本体约需0.1–0.3 km²。配套运营区额外需0.3–1 km²。合计约**0.4–2 km²**。

**纬度偏差的力学影响估算**：

若锚碇点纬度偏差为Δφ（单位：度），水平分力与总张力之比约为：
\[
F_{水平} / T \approx \sin(\Delta\phi) \approx \frac{\Delta\phi \times \pi}{180}
\]

对于Δφ=1°（约111 km），水平分力约为总张力的1.75%（以地面端张力10¹⁰ N量级估算，约1.75×10⁸ N）。

**本卷选址准则**：优先Δφ<0.5°（水平分力<1%）；Δφ<1°可接受；>1°原则上排除，仅在无替代陆地资源时保留。

## 3.3 锚碇方案设计

### 3.3.1 陆基锚碇（节点1、5，纯抗拔型）

**设计原理**：电梯缆绳张力通过倒锥形锚碇井+垂直预应力岩锚群传递至基岩。Δφ≈0°时无水平分力，采用纯抗拔设计。

**工作荷载推导**：由第四卷缆绳设计中的锥形截面模型和张力分布计算得出，地面端最大张力T_max = σ_许用·A₀/S（σ_许用取第二卷2.3.4节目标值~20 GPa，A₀取概览地面端截面面积~0.196 m²，S取第二卷2.7.1节独立工况安全系数≥2.5）。代入得T_max = 20×10⁹×0.196/2.5≈1.57×10⁹ N。在此基础上考虑锚碇安全系数（抗拔）≥3.0，岩锚群需承载的设计荷载≈1.57×10⁹×3.0≈4.71×10⁹ N。

| 参数 | 设计值/范围 | 说明 |
|------|------|------|
| 工作荷载（最大张力） | ~1.57×10⁹ N | 独立工况下地面端最大张力（推导见第四卷缆绳设计张力分布部分） |
| 安全系数（抗拔/缆绳断裂） | ≥3.0 / ≥2.5 | 抗拔系数高于环缆绳的2.0，因岩体力学参数不确定性 |
| 岩锚群数量 | ≥100根（每根30–50 MN承载力） | 按工作荷载×安全系数反算 |
| 锚碇井直径/混凝土用量 | ~25 m / ~8,000 m³ | — |

### 3.3.2 陆基锚碇（节点2、3，抗拔+抗剪复合型）

**设计原理**：在纯抗拔型基础上增加倾斜岩锚（倾斜角≈Δφ×1.3，安全系数）和井壁剪力键，锚碇直径扩大约15%–20%。

| 参数 | 节点2（Δφ=0°43'S） | 节点3（Δφ=0°57'S） | 说明 |
|------|------|------|------|
| 工作荷载（最大张力） | ~1.57×10⁹ N | 同左 | 独立工况下地面端最大张力（推导见第四卷缆绳设计张力分布部分） |
| 安全系数（抗拔/缆绳断裂） | ≥3.0 / ≥2.5 | 同左 | — |
| 水平分力 | ~1.10×10⁸ N | ~1.45×10⁸ N | T_max×sin(Δφ) |
| 水平分力占比 | ~1.26% | ~1.66% | sin(Δφ) |

### 3.3.3 岛屿/深海吸力锚锚碇（节点4，倾斜系留型）

## 3.5 节点与环主体的接口设计

### 3.5.1 环段弧距分布与论证

环半径R₀=42,264 km（见第一卷1.2.1节），周长C=2πR₀≈265,458 km，每度弧长l_度≈737.4 km/°。

各环段弧长：

| 环段 | 弧长 (km) | 占环总长 |
|------|------|------|
| 让蒂尔港→印度洋平台 | ≈47,360 | 17.8% |
| 印度洋平台→勿里洞 | ≈25,810 | 9.7% |
| 勿里洞→圣诞岛 | ≈69,700 | 26.3% |
| 圣诞岛→曼塔 | ≈56,600 | 21.3% |
| 曼塔→马卡帕 | ≈21,920 | 8.3% |
| 马卡帕→让蒂尔港（闭合） | ≈44,080 | 16.6% |

**冗余度验证**：圣诞岛失效时合并弧段约126,300 km（占47.6%，<50%），环不整体解体。

### 3.5.2 机械连接接口——GEO端连接设施完整方案

环建成后（第4阶段），电梯缆绳顶端需从配重空间站切换至环主体。以下从方案对比、选型论证到参数锁定的完整设计过程。

**（1）接口功能需求**

GEO端连接设施必须同时满足以下功能：
- **承载传递**：将缆绳顶端残余张力（T_GEO≈1.24×10⁸ N，见第四卷配重系统设计）安全传递至环缆绳
- **动载荷承受**：电梯运行过程中产生的加减速载荷（约0.05g×200 t≈9.81×10⁴ N）及轿厢通过的周期性交变应力（10⁸次疲劳寿命要求）
- **定位与姿态保持**：缆绳顶端在GEO高度须精确对准环缆绳上的接口点（允许偏差<1 m）
- **维修与更换可达性**：连接件为可拆卸式设计，允许在轨更换（设计寿命≥500年的结构中，连接件本身寿命约50–100年，需多次更换）

**（2）候选方案对比**

| 方案 | 原理 | 优点 | 缺点 | 可行性 |
|------|------|------|------|:--:|
| **方案I：锥形过渡段+预紧螺栓群** | 缆绳顶端通过锥形过渡段与环缆绳接口嵌合，用多根高强度螺栓预紧 | 结构简洁，承载明确，预紧力可控 | 螺栓松动和应力腐蚀风险需长期监测；安装时需高精度对准 | ★★★★ |
| **方案II：锥形套筒+楔形自锁** | 利用缆绳残余张力将锥形套筒楔入环缆绳接口的匹配锥面内 | 无螺栓，无应力腐蚀，随张力增大而自紧 | 拆卸困难；锥面磨损不可逆 | ★★★ |
| **方案III：电磁夹持+机械备份** | 电磁铁提供主要锁紧力，机械销钉提供冗余 | 可远程解锁，便于拆卸 | 电磁铁需持续供电；在GEO辐射环境中线圈老化需考虑 | ★★ |

**推荐方案I**。理由：
- 预紧螺栓群在桥梁主缆锚固和大型悬索结构中已有数十年成熟应用，承载能力和疲劳寿命均满足要求
- 方案II的自锁结构在更换连接件时（50–100年周期）拆卸极为不便，对在轨维修的友好度不足
- 方案III的电磁铁在GEO辐射环境中须额外防护，增加复杂度且不可靠

**（3）方案I详细设计参数**

| 参数 | 设计值 | 推导依据 |
|------|------|------|
| 设计载荷 | ≥3.1×10⁸ N | T_GEO×S（≈1.24×10⁸×2.5），T_GEO见第四卷配重系统设计，S见第二卷2.7.1节 |
| 连接件材料 | C/SiC复合材料 | 许用应力≈500 MPa，耐辐射、抗疲劳、热膨胀系数与CNT缆绳匹配 |
| 过渡段长度 | ~2 m | 从缆绳顶端ø0.14 m锥形过渡至环缆绳接口ø0.3 m（锥度≈4.6°），缆绳顶端直径见第四卷缆绳设计张力分布 |
| 螺栓数量/规格 | 16–20根，M48–M56级 | 单根预紧力≥2×10⁷ N，材料为C/SiC或钛合金，总预紧力≥3.2×10⁸ N |
| 环缆绳局部加强 | 接口两侧各10 km环段增加冗余股数+20% | 局部应力集中补偿（精确值由第五卷仿真确定） |
| 疲劳寿命 | >10⁸次 | 含螺纹咬合面的微动磨损控制在1 μm/10⁶次以内 |

**（4）安装与对准机构**

- 配重空间站（独立工况下）末级配备牵引导向缆，将缆绳顶端从配重接口逐步引导至环缆绳下方约50 m处
- 环缆绳接口处设有**主动抓捕臂**（自由度为3个移动轴+3个旋转轴），精确定位缆绳顶端连接件，对准偏差<1 cm
- 预紧螺栓群由螺栓牵引机器人逐一拧紧至设计扭矩（力矩控制精度±5%）

**（5）环缆绳接口的兼容性处理**

环建成后，环缆绳在6个节点位置预留接口座，缆绳顶端连接件插入接口座的匹配锥槽后完成锁紧。切换过程须在GEO轨道站上由维修机器人完成——配重空间站在第4阶段退役，转为冗余备份或推进至坟墓轨道。
