## 1.1 卷首说明

本卷定位为太空环梯工程的物理地基，聚焦于回答一个根本问题：

> **一条半径42,264公里的闭环缆绳，在仅考虑引力与惯性力的条件下，是否存在力学上的平衡解？平衡的性质是什么？维持平衡需要应对外部摄动的量级如何？**

全卷在推导时不引用任何来自材料科学的强度、线密度、弹性模量等具体数值——这些参数由第二卷提供，并由后续第四卷（综合物理论证）汇合前三卷输入后给出最终的稳定性判定。

本卷的任务是：
1. 建立环的力学模型与平衡方程。
2. 确定环的旋转模式选择及其物理后果。
3. 对被动静平衡问题进行模态分解，给出每种模态稳定性判定的方法论，暂不填入具体数字。
4. 穷举所有摄动源并估算其量级与累积效应。
5. 推导主动维持系统的需求框架。
6. 分析失效模式与安全条件的物理逻辑。

## 1.2 环的定义与坐标系

### 1.2.1 基本几何参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 环半径 R₀ | 42,264 km | 标称值，地球同步轨道（42,164 km）外缘 +100 km |
| 环周长 C | ≈ 265,458 km | C = 2πR₀ |
| 环截面直径 d | 0.5 – 1.0 m | 设计区间，本卷取 0.7 m 为讨论基准 |
| 环线密度 λ | 待第二卷输入 | 取决于截面与材料 |
| 环等效弯曲刚度 | 待第二卷输入 | 取决于编织结构与材料弹性模量 |

> 以上参数中，线密度与弯曲刚度不参与本卷推导，仅在给出公式时以符号保留。

### 1.2.2 坐标系

地心惯性坐标系：原点位于地球质心。Z轴沿地球自转轴指向北极，X轴指向春分点，Y轴由右手定则确定。环理想状态下位于赤道平面（XY平面），赤道面即地球赤道面。

环上一点P的位置由角坐标φ（从X轴起算）唯一确定。

### 1.2.3 摄动项约定

后续所有涉及引力场的计算，地球引力采用：

- 中心项：\( V_0 = -\frac{GM_e}{r} \)
- J₂摄动项：\( V_{J_2} = -\frac{GM_e}{r} J_2 (R_e/r)^2 \frac{3\sin^2\phi_{lat}-1}{2} \)（赤道面上 \(\phi_{lat}=0\)）

月球与太阳作为质点引力源处理。

## 1.3 理想静平衡

### 1.3.1 无摄动平衡条件

忽略一切摄动，假设地球为中心引力场，环为连续体。取环上一微元（对应角坐标dφ），受力：

- **引力**：\( dF_g = -\frac{GM_e \lambda}{R_0} d\phi \hat{r} \)
- **张力**：两端张力T的径向合力为 \( dF_T = T d\phi \hat{r} \)（向中心为正）

平衡方程：
\[
T = \frac{GM_e \lambda}{R_0}
\]

此式说明：对于给定线密度λ与半径R₀，存在唯一确定的张力T使环在径向受力平衡。平衡不依赖于任何材料假设——只要缆绳能承受此张力，环即可存在。

### 1.3.2 同步轨道外缘的特殊性

在R₀ = 42,264 km处：
- 当地引力加速度：\( a_g = \frac{GM_e}{R_0^2} \approx 0.2238 \) m/s²
- 以地球自转角速度 ω_e = 7.2921×10⁻⁵ rad/s 同步旋转所需的向心加速度：\( a_c = \omega_e^2 R_0 \approx 0.2247 \) m/s²

\( a_c > a_g \)，差值 ≈ 0.0009 m/s²。

**含义**：若环仅受引力，其向心加速度仅为 a_g，不足以维持半径为 R₀ 的同步圆周运动。缆绳必须提供额外的向心拉力来补足此差值，使净向心加速度达到 a_c。环因此处于永久受拉状态，储存弹性能。

### 1.3.3 两种旋转模式

|  | 模式一：同步旋转 | 模式二：开普勒轨道速度 |
|------|------|------|
| 环的角速度 | ω_e | ω_开普勒 ≈ ω_e + Δω |
| 环相对地面 | 静止 | 切向滑动 ~17 m/s |
| 缆绳张力 | 有（约需弥补向心力差） | 零（引力 = 向心力） |
| 电梯锚碇 | 固定节点，无切向摩擦 | 锚碇需持续承受环滑动力 |
| 概览意图 | 是 | 否 |

**本卷选择**：按概览设计意图，采用**模式一（同步旋转）**。后续所有摄动分析、模态分析及主动控制推导均基于同步旋转环。

## 1.4 被动静平衡的模态分解

### 1.4.1 问题重述

理想圆环的平衡是否自稳定？若非自稳定，哪些模态需要主动控制？

### 1.4.2 扰动展开

环偏离标称半径的径向位移 δr(φ, t) 展开为：
\[
\delta r(\phi, t) = \sum_{n=0}^{\infty} [A_n(t) \cos(n\phi) + B_n(t) \sin(n\phi)]
\]

### 1.4.3 各模态行为

| 模态阶数 | 形态 | 恢复机制 | 初步判断 | 判定方法 |
|------|------|------|------|------|
| n=0 | 均匀膨胀/收缩 | 重力梯度 | 可能不稳定 | 推导径向摄动方程，检查恢复力项符号。第四卷代入λ后计算。 |
| n=1 | 质心平动 | 无恢复力（零频） | 中性稳定 | 对应动量守恒。对环完整性无直接威胁，周期校正即可。 |
| n≥2 | 波浪形变形 | 弯曲刚度 + 张力 | 短波稳定，长波待定 | 求解弹性环频散关系，得到临界波长。第四卷代入弯曲刚度后计算。 |

**关于n=0模不稳定性的补充说明**：定性而言，若环略微向外膨胀，引力减小（∝ 1/R²），但同步旋转所需的向心加速度变化规律与引力变化并不相同——膨胀后环需要更大的缆绳拉力来维持平衡，而这又会进一步拉伸环，形成正反馈。因此n=0模本质上有发散倾向，需由主动控制系统持续抑制。

### 1.4.4 判定框架（提供给第四卷）

- **n=0模判定方程**：\( \ddot{A}_0 + (\omega_{n=0})^2 A_0 = 0 \) —— 若 ω² > 0 则稳定，< 0 则指数发散。ω²的表达式含λ，由第四卷代入数值计算。
- **n≥2模频散关系**：\( \omega^2(n) = f(n, T, 弯曲刚度, λ) \) —— 若对所有n有 ω² > 0 则被动稳定；若存在 n > 某临界值时 ω² < 0，则该模态需主动阻尼。

## 1.5 摄动分析

### 1.5.1 摄动源总表

| 摄动源 | 周期性 | 量级（×地表重力加速度） | 主要效应 |
|------|------|------|------|
| 月球引力 | 月周期 27.3天 | ~10⁻⁶ | 倾覆力矩，环面倾角漂移 |
| 太阳引力 | 年周期 | ~0.5×10⁻⁶ | 倾角年变化，质心偏移 |
| 地球扁率 J₂ | 长期 | ~10⁻⁶（同步轨道外） | 环面升交点赤经进动 |
| 太阳光压 | 连续 | ~10⁻⁹（光压 ~9×10⁻⁶ 牛/平方米）[1.2] | 质心偏移累积 |
| 月球轨道偏心率 | 月周期调制 | ~10⁻⁷ | 潮汐力波动 |

> 注：J₄项及更高阶带谐项在同步轨道高度的效应约10⁻⁹–10⁻¹⁰量级，较上述摄动源弱至少三个量级，初步论证阶段不予纳入。可在第四卷数值仿真中作为灵敏度分析的可选项加入。

### 1.5.2 月球摄动（核心项）

月球绕地轨道面与赤道面夹角在18.3°–28.6°之间变化（周期约18.6年）[1.1]。月球对环上任一点的引力在垂直赤道面方向的分量，在环的不同经度处符号相反，产生倾覆力矩。

**所需计算**（由第四卷完成）：
- 环面倾角的时间导数与积分值（跨度≥10年）
- 倾覆力矩的傅里叶谱（识别主要周期成分）
- 抵消此力矩所需的推进器推力分布

初步量级估计：倾角年漂移率约0.01°–0.1°。

### 1.5.3 太阳引力摄动

效应与月球摄动相似但幅值较小（约月球的1/3–1/2），叠加年周期。

### 1.5.4 地球J₂项

J₂项引起环面升交点赤经进动。在同步轨道高度此效应弱于近地轨道约三个量级，每百年数角秒量级 [1.3]。对环本身稳定无实质威胁，仅影响对地通信天线指向校准。

### 1.5.5 太阳光压

环在平行光方向的有效投影面积 ≈ 2R₀ × d ≈ 5.9×10⁷ 平方米（取 d = 0.7 m）。光压约9×10⁻⁶ 牛/平方米（太阳常数取1361 瓦/平方米，完全反射条件下）[1.2]，总力约530牛，连续均匀作用。数月内可累积出可观测的质心偏移（n=1模），需周期抵消。

## 1.6 主动维持系统需求框架

### 1.6.1 任务分解

| 任务 | 所需推力特征 | 触发频率 |
|------|------|------|
| 半径控制（n=0） | 连续微调 | 连续 |
| 质心保持（n=1） | 周期冲量抵消光压累积 | 每数日一次 |
| 倾角维持（月球+太阳） | 周期力矩抵消 | 按月/年尺度 |
| 局部应力调整 | 按需 | 随机 |

> 注：n=1模质心校正的单次冲量量级取决于具体的校正频率与光压累积时长。以530牛光压连续作用一日为例，累积动量约4.6×10⁷牛·秒；若校正频率为数日一次，冲量将进一步增大。精确冲量与对应推进剂消耗率等第四卷仿真确定。

### 1.6.2 推进器配置（初步框架）

- 数量：≥ 36 个（每10°一个，保证控制自由度覆盖n≥2至合理阶数）
- 总推力容量：~100–200 牛量级
- 推进剂类型：离子推进器（氙/碘），比冲约3000秒
- 能源需求：约数百千瓦（由后续能源卷详述）

**精确推力上限与推进剂消耗率**由第四卷仿真结果确定。

## 1.7 失效模式与安全逻辑

### 1.7.1 单股断裂

设环由N股独立缆绳冗余编织。单股断裂后，其原承载张力在剩余N-1股间重分配。当N较大（概览取N≥1000），单股断裂引起的应力增量<0.1%，不构成整体威胁。

### 1.7.2 多股同时断裂

若碎片撞击导致某段M股同时断裂，当M/N超过某一临界值时，剩余股无法承受环张力，该段将完全断裂。临界比值取决于安全系数设计。

**安全准则**：任意10公里环段内可承受的瞬时股断裂数上限取决于实际安全系数设计值，该值由第二卷提供后，在第四卷中确定。

> 推导示例：若安全系数取2.0，则全截面强度为工作应力的2倍。断掉一半（0.5N）后剩余股应力翻倍，恰好达到极限强度。因此0.5N为该安全系数下的理论断裂上限。安全系数3.0时，理论上限升至约0.67N，以此类推。实际设计将取比理论上限更保守的值（例如安全系数2.0时取0.3N），以涵盖材料不均匀性与应力集中等因素。

本节给出的具体比例为推导示例，并非最终设计值。实际可承受比例须在第四卷中结合第二卷截面参数与材料实测数据综合确定。

### 1.7.3 电梯缆绳断裂

单座电梯缆绳断裂不影响环的闭环完整性。环可在剩余节点间重新分配载荷。断裂节点需重建缆绳后重新接入，期间该节点停用。

## 1.9 第一卷结语

本卷完成了环梯的物理基础框架搭建。核心结论：

1. 同步旋转环在赤道面内的静力平衡方程成立。环处于永久受拉状态，张力与线密度成正比。
2. 被动静平衡的模态已分解为n=0、n=1、n≥2三类，每类的稳定性判定方程已给出，等待第二卷的参数输入后在第四卷完成定量判定。
3. 主要摄动源（月球、太阳、J₂、光压）已穷举并给出量级。月球倾覆力矩是最核心的长期扰动，倾角漂移率初步估计为每年0.01°–0.1°。
4. 主动维持系统框架已提出（≥36个推进器，总推力~100–200牛），精确参数等第四卷仿真锁定。n=1模质心校正的冲量量级已给出估算方法与范围，具体数值待校正策略确定后填入。
5. 失效模式的安全逻辑已建立，临界条件待第四卷定量化。

**当前状态**：第一卷自身已可独立结案。无需等待第二卷即可审阅本卷的逻辑严密性与力学框架完整性。所有数字上的不确定性均已转化为明确的求解路径，移交第四卷。
